# Parry Marketing Website

A simple, modern marketing website for Parry Security Scanner.

## Features

- Clean, modern design with gradient hero
- Responsive layout for mobile and desktop
- Feature highlights and statistics
- Comparison table with competitors
- Quick installation guide
- No JavaScript required (pure HTML/CSS)

## Deployment

### GitHub Pages
1. Push the `website/` folder to your repository
2. Enable GitHub Pages in repository settings
3. Point to `/website` directory

### Netlify
1. Drag and drop the `website/` folder to Netlify
2. Or connect your repository for auto-deploy

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the `website/` directory

### Apache/Nginx
Just serve the `index.html` file as the root document.

## Customization

Edit `index.html` to customize:
- Colors: Change gradient values in `<header>` and `.stats`
- Content: Update feature descriptions and statistics
- Links: Modify footer links and CTA buttons
